# CraftHub简介

### 📋 项目简介

​	CraftHub是一个简单的集成GUI框架， 内嵌多个简单好用的脚本，支持持续扩展，致力于减少用户消耗在重复工作上的时间。



### ✨ 主要特性

​	\- ✅ GUI集成：将命令行脚本转换为GUI格式

​	\- ✅ 可扩展性：可扩展添加新的脚本

​	\- ✅ 集成环境：工具无需配置环境，下载即用



### 📦 目前工具

​	\- ✅ AutoRoute: 路由计算工具

​	\- ✅ SplitImage: 工勘照片分类工具
